# Indiadex

**Buy & sell crypto globally with local currency.**

[indiadexswap.xyz](https://indiadexswap.xyz) — A fiat-to-crypto on-ramp/off-ramp aggregator built on regulated infrastructure, supporting 40+ countries with local payment methods.

## Features

- **Buy & Sell Crypto** — 100+ cryptocurrencies via Transak (regulated KYC/AML partner)
- **Cross-Chain Swap** — Powered by LI.FI, supporting 20+ blockchains
- **Multi-Currency** — INR, USD, EUR, GBP, AUD, CAD, SGD, AED, JPY, BRL and more
- **Local Payments** — UPI, IMPS, NEFT, SEPA, debit/credit cards, Apple Pay, Google Pay, bank transfers
- **Non-Custodial** — Crypto sent directly to user's wallet
- **Competitive Fees** — Transparent pricing with no hidden charges

## Architecture

- **Next.js 16** with Turbopack
- **Tailwind CSS** for styling
- **Transak SDK** for fiat on-ramp/off-ramp
- **LI.FI Widget** for cross-chain swaps
- **Onramp.money** as backup provider
- **Vercel** for hosting and deployment

## Deployment
